# Copy Code Snippet Box

This project demonstrates a clean, interactive **code snippet box** with a "Copy to Clipboard" button.  
When the user clicks the button, the code inside the box is copied to the clipboard, and the button text updates to indicate success or failure.

## Features
- Copy code to clipboard with one click
- Button shows **"Copied!"** on success and reverts after 2 seconds
- Error handling for failed copy attempts
- Responsive and visually appealing design

## Technologies Used
- HTML5
- CSS3 (flexbox, gradients, transitions)
- JavaScript (Clipboard API, event handling)
